Projeto de seleção para a empresa TEC FUN

Devido a problemas de conexão com o banco de dados do PythonAnywhere, a plicação encontra-se sem banco de dados, todas as configurações estão corretas e de acordo com os padrões do servidor. Problemas na aplicação podem ocorrer proveniente disso.

A url principal da aplicação é /main que só pode ser acessada por usuários logados. Os usuários e senhas desta aplicação são, admin - secret e test - test

O Read eDelete do CRUD, se dá passando um id por meio das urls /read/ e /delete/
